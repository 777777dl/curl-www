CURLHandle is a Cocoa subclass of NSURLHandle that wraps an object-oriented shell around libcurl's C interface.

How to Install
Include CURLHandle.[hm], and (optionally) CURLHandle+extras.[hm] in your project

Examples
CURLHandleTester is an example that comes with the distribution.

Binding documentation
View it on the CURLHandle web site
