We also have this list of programs using libcurl. TITLE(curl-config is our friend)

Applications (or actually whoever) that want to find out about the libcurl installation on a particular host, will find a friend in the curl-config tool. (Added in the curl 7.7.2 release.)

What is the path to the curl header files? 

 echo `curl-config --prefix`/include

What is the path to libcurl ? 

 echo `curl-config --prefix`/lib

What linker options do I need when I link with libcurl? 

 curl-config --libs
(you will also need -lcurl for libcurl itself and then possibly a -Llibpath to curl's library path)

How do I know if libcurl was built with SSL support? 

 curl-config --feature | grep SSL

What's the installed libcurl version? 

 curl-config --version
