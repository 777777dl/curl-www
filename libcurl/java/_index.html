libcurl is a solid and reliable multi-protocol file transfer library for URL syntax defined files. We can take advantage of libcurl straight from within our Java programs.

All you need to do is to install the 'javacurl' package and then you'll be able to use it.

This interface is still very basic. We need your input and suggestions on how to develop this further!

How to Install
Install curl and libcurl first. Manual hands-on installation is required for the actual java interface module.
