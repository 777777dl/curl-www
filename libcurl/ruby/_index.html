libcurl is a solid and reliable multi-protocol file transfer library for URL syntax defined files. We can take advantage of libcurl straight from within our Ruby programs.

All you need to do is to install the Ruby/libcurl package and then you'll be able to use it.

Visit the Ruby/libcurl homepage.

How to Install
