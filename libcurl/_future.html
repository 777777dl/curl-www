We need to add a new interface to libcurl, an interface that allows the calling application to "drive". We call that a "pull" interface.

This was once upon the time dicussed, calling it a "low level" interface. It was brought up again during the autumn 2001 and we have now refined the ideas and our thoughts are now gathered around the "multi" interface.

We have a simple, imaginary multi.h header file, and an associated similarly imaginary multi-app.c application source code, showing how this interface could be made to work.

No code has yet been written to make this come real.

Bring your ideas and energy to the libcurl mailing list! #include "_footer.html"
