We intend to add a new interface to libcurl, an interface that allows the calling application to "drive". We call that a "pull" interface.

This was once upon the time dicussed, calling it a "low level" interface. It was brought up again during the autumn 2001 and we have now refined our ideas and our thoughts are now gathered around the "multi" interface. SUBTITLE(header)

We have started working on the multi.h header file, which also pretty much contains all the docs there is yet. SUBTITLE(source)

The multi.c file is to be filled in with source code. SUBTITLE(example using this interface)

The imaginary multi-app.c is application source code, showing how this interface could be used.

Bring your ideas and energy to the libcurl mailing list! #include "_footer.html"
