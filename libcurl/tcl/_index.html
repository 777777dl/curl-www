libcurl is a solid and reliable multi-protocol file transfer library for URL syntax defined files. We can take advantage of libcurl straight from within our Tcl programs.

All you need to do is to install the TclCurl package and then you'll be able to use it.

Visit the TclCurl homepage.

How to Install
Install curl and libcurl first, then follow the usual sequence: 

  ./configure
  make
  make install
To test the installation do: 
 $tclsh
    % package require TclCurl

If Tcl returns TclCurl's version number, all went well.

Examples
One basic example: 

 package require TclCurl
    set curlHandle [curl::init]
    $curlHandle setopt CURLOPT_URL http://curl.haxx.se
    $curlHandle setopt CURLOPT_FILE index.html
    $curlHandle perform
    $curlHandle cleanup
There are more examples on the TclCurl site
