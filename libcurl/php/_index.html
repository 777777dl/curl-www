libcurl is a solid and reliable multi-protocol file transfer library for URL syntax defined files. We can take advantage of libcurl straight from within our PHP programs. Curl is supported built-in since PHP 4.0.2, no extra stuff is needed but the PHP and curl packages.

All you need to do is to configure PHP with curl awareness on build time, and you'll get a set of extra functions to use.

How to Install?
You need to enable the curl support when you build PHP: install the curl-PHP interface.

Application
We collect a list of PHP applications powered by curl.
