We can take advantage of libcurl straight from within PHP programs. Curl is supported built-in since PHP 4.0.2, no extra stuff is needed but the PHP and curl packages.

All you need to do is to configure PHP with curl awareness on build time, and you'll get a set of extra functions to use.

How to Install?
You need to enable the curl support when you build PHP: install the curl-PHP binding. There's also the document with hints on how to install PHP/CURL on IIS.

Books
So far, there aren't many books for PHP/CURL only, but the following books have a varying degree of PHP/CURL documentation included:

  • PHP Functions Essential Reference by Zak Greant, Graeme Merrall, Torben Wilson and Brett Michlitsch. It has a full chapter with basicly the PHP/curl online manual included.
  • The PHP Programmer's Cookbook by Sterling Hughes contains some curl-using examples. (out of print now, rumours mention a 2nd edition in the future)
  • PHP and MySQL Web Development by Luke Welling and Laura Thomson contains some curl info.
  • Beginning PHP4 by Lea, Choi, Ken, Prasad and Ullman seems to only mention a few curl functions very briefly.
  • CURL PHP Examples Book with Tutorial Help and online Assistance by Muhammad Imran Khalid. It is claimed to contain examples, step by step guides, "header tracking solutions", XML, credit card processing etc.
