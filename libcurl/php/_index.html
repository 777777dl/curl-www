We can take advantage of libcurl straight from within PHP programs. PHP/CURL is supported built-in since PHP 4.0.2, no extra stuff is needed but PHP and libcurl installed.

PHP/CURL is a binding that uses libcurl. It means that the PHP team has written a glue layer for PHP that speaks to the underlying libcurl. That layer, the binding, is what we call PHP/CURL and that is what offers the functions named curl_* within the PHP language for you. The PHP/CURL binding is written, maintained and being provided by the PHP team and you will not find any binaries, DLLs or other packages for it on this site.

How to Install?
The primary site for all things PHP is of course the www.php.net site. See their curl section for details on the PHP/CURL binding.

(If you for some reason still don't read the instructions on the php.net site...) You need to enable the curl support when you build PHP: install the curl-PHP binding. There's also the document with hints on how to install PHP/CURL on IIS.

Books
So far, there aren't many books for PHP/CURL only, but the following books have a varying degree of PHP/CURL documentation included:

  • PHP Functions Essential Reference by Zak Greant, Graeme Merrall, Torben Wilson and Brett Michlitsch. It has a full chapter with basicly the PHP/curl online manual included.
  • The PHP Programmer's Cookbook by Sterling Hughes contains some curl-using examples.
  • PHP and MySQL Web Development by Luke Welling and Laura Thomson contains some curl info.
  • Beginning PHP4 by Lea, Choi, Ken, Prasad and Ullman seems to only mention a few curl functions very briefly.
  • CURL PHP Examples Book with Tutorial Help and online Assistance by Muhammad Imran Khalid. It is claimed to contain examples, step by step guides, "header tracking solutions", XML, credit card processing etc.
