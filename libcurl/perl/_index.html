libcurl is a solid and reliable multi-protocol file transfer library for URL syntax defined files. We can take advantage of libcurl straight from within our Perl programs.

All you need to do is to install the Curl::easy package and then you'll be able to use the Perl interface.

How to Install
Install curl and libcurl first. Make sure you install the libcurl.so library and curl's include files as well. (E.g. if you are using an rpm version of curl, you will need to install the curl-devel rpm before you can build the Perl interface.)

Run these commands: 

   cd [curl::easy-directory]
   perl Makefile.PL
   make
   make test
   make install

Download
Curl::easy 1.1.8 (14Kb) was released on September 20, 2001.

This release includes support for libcurl options in libcurl 7.9, though it can also be rebuilt for earlier libcurl versions. Better test scripts are also included.

There is no need to upgrade from 1.1.6 or 1.1.7 unless you want access to new options in libcurl 7.9, such as SSL certificate name checking.

Curl::easy 1.1.7 (12Kb) was released on September 13, 2001.

If you are using curl 7.8.1 or later, you will need to be using at least version 1.1.6 of Curl-easy.

1.1.7 is a documentation only update, with more explicit licence information. There is no need to upgrade if you are already using 1.1.6.

See Changes for complete change information.

For curl versions before 7.7, download the Perl interface version 1.0.2 from www.koblenz-net.de/~horn/export/

Browse the /perl/Curl_easy/ files that are presently in CVS.
