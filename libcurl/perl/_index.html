libcurl is a solid and reliable multi-protocol file transfer library for URL syntax defined files. We can take advantage of libcurl straight from within our Perl programs.

All you need to do is to install the Curl::easy package and then you'll be able to use the Perl interface.

How to Install
Install curl and libcurl first. Make sure you install the libcurl.so library and curl's include files as well.

Run these commands: 

   cd [curl::easy-directory]
   perl Makefile.PL
   make
   make test
   make install

Examples
Please post any examples you've made, as I have none!

Interface documentation
The interface closely resembles the C one, so it should be OK to study the regular libcurl C/C++ API docs.

Download
The Curl::easy interface is part of the full source code release archive from version 7.7 and onwards. It is however also released separately and thus there may be a newer perl interface. Check out www.koblenz-net.de/~horn/export/ for the latest packages.

curl::easy 1.1.5 (12Kb) was uploaded on April 20, 2001.

For curl versions before 7.7, download Perl interface version 1.0.2

Browse the /perl/Curl_easy/ files that are presently in CVS.
