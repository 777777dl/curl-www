OS/Distributor OS version

Lots of other Linux distributions are based on either the Debian or the Redhat distributions, so therefore you're likely to find curl packages in a lot of other environments as well.

If you use a curl version you got bundled with your OS distribution, it is very likely that you don't have the latest version. This may not matter, but if you find anything you believe is a bug, consider upgrading and try the same operations on a newer release first! #include "_footer.html"
