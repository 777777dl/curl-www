Roland Krikava provided this very useful script that builds a CA cert bundle useful for curl, by extracting information provided in the Mozilla source tree. The Mozilla file needed for this job is named /mozilla/security/nss/lib/ckfw/builtins/certdata.txt.

parse-certs.sh is a shell script with an embedded Ruby script. You thus need to have Ruby installed to make this work.

Note: the script uses the 'openssl' tool and it requires the version that comes with the 0.9.7 series. SUBTITLE(We Did This For You)

We run a script regularly that makes this conversion automatically. The output CA bundle file in PEM format is available from here: cacert.pem (~320KB) and cacert.pem.gz (~113 KB).

The output file contains the datestamp of the conversion. SUBTITLE(License)
