The lib/mk-ca-bundle.pl script (in CVS and bundled in curl tarballs) converts Mozilla's CA cert bundle to PEM format, suitable for (lib)curl and others. Writtten by Guenter Knauf. SUBTITLE(Automated Conversions)

We provide automated conversions - done weekly. The output CA bundle file in PEM format is available from here:

  • cacert.pem (220KB)
  • cacert.pem.gz (110 KB)
  • cacert.pem.bz2 (110 KB)

The file contains the datestamp of the conversion. SUBTITLE(Requirements) It is a perl script. The script uses the 'openssl' tool and it requires the version that comes with the 0.9.7 series or later.

The exact Mozilla file needed for this job is found within that script. SUBTITLE(License)
