This web site and associated mirrors are dedicated to covering curl, libcurl and related issues only.

If you have any complaints, ideas or suggestions for this web site, don't hesitate to mail feedback to curl-web at haxx.se. We do make a serious effort to please our audience. This email address will always reach the right person.

The curl web site contents are...

  • maintained and edited by Daniel Stenberg. If you wanna join in and help out with this, mail us! Dan Fandrich helps me keep the download page in shape.
  • mirrored to the US by Japheth Cleaver and Eric L
  • mirrored to Australia by Jason Andrade of planetmirror.com
  • mirrored to the Russan Federation by Alex Andrienko of Tsuren.net
  • mirrored to Germany by Sven Wegener
  • mirrored to Thailand by Taweewit Pensawat

The box running curl.haxx.se is...

  • hosted by Contactor Data AB
  • running Apache 1.3.x on Linux
  • hooked to a 2 Mbit/sec connection
  • using a subdomain provided by Haxx
  • serving more than 50000 unique visitors that download more than 15GB per month

The main technologies used when creating this site are:

  • Daniel's cpp that all static pages are preprocessed with when generated
  • standard unix make for dependency-stuff
  • perl for dynamic/CGI pages and most background scripting
  • hypermail for mail archives
  • LXR for source browsing
  • man pages converted using roffit
  • txt2html by Seth Golub converts some plain-text files to HTML
  • curl(!) for the automaticly generated mirror links
  • contents are managed and distributed using CVS (CVS repository hosted by php.net)

We hardly display any email addresses on this site without obscuring them by removing their '@' sign. Research has shown that when we publish addresses in plaintext, we get a whole lot more spam. Sorry for the inconvenience.

The bug tracker is provided by sourceforge.net for the benefit of curl users.
