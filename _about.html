This web site is dedicated to covering curl, libcurl and related issues only.

If you have any complaints, ideas or suggestions for this web site, don't hesitate to mail feedback to curl-web@haxx.se or post it in the feedback forum. We do make a serious effort to please our audience.

If you have a question regarding curl or a curl-related issue, please use a proper curl mailing list.

The box running the main web site curl.haxx.se is...

  • hosted by Contactor Data AB
  • running Apache on Debian Linux
  • hooked to a 2 Mbit/sec connection
  • using a subdomain provided by Haxx
  • serving 100,000 unique visitors that download more than 50GB per month
  • physically located in Akalla (northern Stockholm), Sweden

Technologies used:

  • Daniel's fcpp that most HTML pages are preprocessed with.
  • standard make(1) for dependency-stuff
  • perl for dynamic pages and most background scripting
  • hypermail for mail archives
  • LXR for source browsing
  • man pages converted using Daniel's roffit
  • curl(!) for the automatically generated mirror links
  • content is managed using CVS
  • mirroring is done with rsync
  • ads by Google, Pricerunner and others. Contact curl-web@haxx.se to discuss getting your ads seen by more than 100,000 unique visistors per month.

We display many email addresses on this site by replacing the '@' sign with a textual version. Research has shown that when we publish addresses in plaintext, we get a whole lot more spam. Sorry for the inconvenience.

The bug tracker is provided by sourceforge.net for the benefit of curl users.

If someone is interested and capable of hosting a good bug tracking system for this project, then please let us know. We'll keep using the sourceforge one until we find a better one.
