There are many different curl mailing lists:

  1. main curl mailing list
      for all main concerns, bugs and talks around the curl command line tool (subscribe to curl-main here)
  2. announcement list
      that only receives announcements of new versions. (subscribe to curl-announce here)
  3. developing using libcurl
      should be of interest for application or interface developers that use libcurl (subscribe to curl-library here)
  4. using curl with PHP
      PHP developers that use the CURL module might find this an interesting place (subscribe to curl-and-php here)
  5. curl source code CVS commits
      receives notifications on all CVS commits done to the curl source module (subscribe to curl-commits here)
  6. curl WWW CVS commits
      receives notifications on all CVS commits done to the curl www module (subscribe to curl-www-commits here)
