Curl depends on external libraries for some features (and libcurl for all features). You can build curl without them, but curl gets a lot better if you have these libraries:


www.openssl.org 		SSL (https) support. Use OpenSSL. Curl compiles, builds and runs perfectly with it.

www.openldap.org 		The LDAP support depends on OpenLDAP.
kerberos4 The kerberos4 support depends on a kerberos4 package! :-) More information can be found at http://www.contrib.andrew.cmu.edu/~shadow/kerberos.html or by asking google.
